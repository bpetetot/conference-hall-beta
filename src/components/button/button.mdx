## Simple

Button without padding, border and background. The simpliest way to display it. Useful to display an icon button, like modals closing cross.

<Playground>
  <Button simple>Simple button</Button>
</Playground>

## Link

A hyperlink with a button style, just by setting `href` prop instead of `onClick`.

<Playground>
  <Button href="https://www.google.fr" target="NEW">Link like a button</Button>
</Playground>

## Using Function As A Child

Used to apply button style to any element.

<Playground>
  <Button secondary accent>
    {btnClasses => <div className={btnClasses}>Apply button style to any element</div>}
  </Button>
</Playground>
